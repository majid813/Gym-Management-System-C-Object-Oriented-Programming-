Gym Management System (C++)
Overview

This project is a console-based Gym Management System developed in C++ using Object-Oriented Programming principles. It helps manage gym member records and basic operations in an organized and efficient way.

Features
Add new members with details
Update existing member records
Delete member records
Search for members
Membership plan management
File handling for data storage and retrieval
Concepts Used
Object-Oriented Programming (Classes, Encapsulation)
File Handling using fstream
Input validation
Modular programming
How It Works

The system stores member data in files to ensure persistence. Users can perform different operations like adding or modifying records, and the system updates the stored data accordingly. The program is designed using OOP to keep the code structured and maintainable.

How to Run
Open the project in a C++ compiler (e.g., Code::Blocks, Dev C++, or VS Code)
Compile the main file
Run the program
Follow on-screen instructions to use the system
